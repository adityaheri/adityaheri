MANDATORY ACCESS CONTROL

Sistem operasi di MAC akan memberikan akses kepada pengguna berdasarkan identitas dan data mereka. Untuk mendapatkan akses, pengguna harus mengirimkan informasi pribadi mereka. Ini sangat aman karena aturan dan batasan diberlakukan oleh admin dan akan diikuti dengan ketat. Pengaturan MAC dan manajemen kebijakan akan dibuat dalam jaringan yang aman dan terbatas pada administrator sistem.

Seperti contoh yang saya terapkan pada mysql, root sebagai admin dapat memberikan izin pada suatu user, izin yang diberikan juga dipengaruhi dari kemauan sang admin itu sendiri.

Disini saya menggunakan user root untuk memberikan privileges select untuk user staf2

DISCREATIONARY ACCESS CONTROL

DAC adalah kontrol akses berbasis identitas. Mekanisme DAC akan dikendalikan oleh identifikasi pengguna seperti username dan password. DAC bersifat diskresioner karena pemilik dapat mentransfer objek atau informasi yang diautentikasi ke pengguna lain. Dengan kata sederhana, pemilik dapat menentukan hak akses.

Seperti contoh yang saya terapkan pada mysql, saya memberikan semua izin pada user staf1, tetapi untuk menggunakan staf1 kita perlu password untuk mengaksesnya

Disini saya menggunakan user root untuk membuat user staf1 dengan password staf1
