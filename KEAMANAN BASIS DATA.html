1. Menyiapkan Web Server
Kita memerlukan sebuah server untuk menjalankan website. Saya menggunakan virtual machine dengan os ubuntu 64-bit sebagai server. Virtual machine ini saya jalankan dengan software vmware

2. Lakukan Update pada Ubuntu

PERBEDAAN UNION, SELECT, WAIT(SLEEP) DAN --

1. UNION

Berfungsi untuk menambahkan baris dari satu table ke table lain

Sebelum melakukan union

Setelah kita melakukan union pada kolom nama

2. SELECT

Select berfungsi menseleksi pilihan

jika kita select nama from mahasiswa maka hanya akan menampilkan kolom nama dari table

3. WAIT / SLEEP

Wait berfungsi untuk memberhentikan database sesuai waktu yg kita inginkan

Berikut contoh jika kita melakukan perintah “do sleep(10); select …” sistem akan berhenti selama 10 detik sebelum melakukan select

4.  --

-- (double dash) berfungsi untuk melakukan comment, layaknya “#” di python dan “<!-- >” di html, fungsi ini tidak melakukan apa pun selain menampilkan karakter yg kita ketik
