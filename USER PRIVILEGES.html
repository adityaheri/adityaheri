PRIVILEGES

Setiap user melakukan perubahan pada database sesuai dengan izin yang diberikan, istilah izin dalam mysql disebut dengan privileges, kita dapat menambahkan izin yang kita mau pada user pilihan kita.

Privileges paling sering digunakan:

·         ALL PRIVILEGES – User yang diberikan privileges ini dapat menggunakan semua privileges

·         CREATE – User yang diberikan privileges ini dapat membuat database, table atau user .

·         DROP – User yang diberikan privileges ini dapat melakukan drop(menghapus database atau table).

·         DELETE - User yang diberikan privileges ini dapat menghapus baris pada table.

·         INSERT - User yang diberikan privileges ini dapat membuat baris pada table.

·         SELECT – User yang diberikan privileges ini dapat melihat isi table.

·         UPDATE - User yang diberikan privileges ini dapat memperbarui baris dalam table.
